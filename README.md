# Calculadora de Média de Alunos

## Descrição
Projeto desenvolvido em linguagem C para calcular a média de duas notas e informar se o aluno foi aprovado ou reprovado.

## Tecnologias Utilizadas
- Linguagem C
- GCC

## Funcionalidades
- Receber duas notas
- Calcular média
- Informar aprovação ou reprovação

## Contexto
Projeto desenvolvido para a disciplina de programação.
